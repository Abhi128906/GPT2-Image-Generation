# Image Generation using Stable Diffusion

## Objective
Generate AI-based images from text prompts using a pre-trained Stable Diffusion model.

## Technologies Used
- Python
- Google Colab
- Hugging Face Diffusers
- PyTorch

## Features
- Text-to-image generation
- Uses Stable Diffusion pre-trained model
- Generates realistic AI images

## Sample Prompt
"A futuristic city at night with neon lights"

## Outcome
Successfully generated AI-based images from text prompts.
